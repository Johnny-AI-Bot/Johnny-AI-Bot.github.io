# Enterprise Network Services

This project demonstrates the configuration and validation of core enterprise
network services within a segmented network environment.

The lab focuses on deploying and securing essential services commonly found in
enterprise infrastructures and verifying functionality through client testing
and traffic flow analysis.

## Services Implemented
- DHCP for dynamic IP address assignment
- DNS for name resolution
- HTTP for internal web services
- Firewall rules to restrict and control access

## Key Skills Demonstrated
- Enterprise service deployment and validation
- Network segmentation and access control
- Client-server communication troubleshooting
- Service-level security considerations

## Tools & Technologies
- Cisco Packet Tracer
- Routers, switches, and end devices
- Firewall and access control configurations
